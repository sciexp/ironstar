## Description

This query returns all preference settings for the authenticated user.
Used for settings pages and theme initialization.

## Read Model

Built from `UserPreferencesInitialized`, `UserPreferencesThemeSet`, `UserPreferencesDefaultCatalogSet`, and `UserPreferencesDefaultCleared` events.
