## Description

This event is emitted when a user sets their default catalog.
The default catalog is automatically selected when starting new sessions.

## Business Rules

- Catalog must be available at the time of setting
- Fallback behavior is implemented if catalog becomes unavailable
- Only one default catalog per user

## Given-When-Then

Given preferences exist and catalogs are available, When the Data Analyst sets a default catalog, Then User Preferences Default Catalog Set is recorded.
