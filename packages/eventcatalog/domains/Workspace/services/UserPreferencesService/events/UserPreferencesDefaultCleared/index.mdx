## Description

This event is emitted when a user removes their default catalog setting.
After clearing, the user will be prompted to choose a catalog each session.

## Business Rules

- A default catalog must currently be set
- Clearing is always allowed
- User will see catalog selection on next login

## Given-When-Then

Given a default catalog is currently set, When the Data Analyst clears the default, Then User Preferences Default Cleared is recorded.
