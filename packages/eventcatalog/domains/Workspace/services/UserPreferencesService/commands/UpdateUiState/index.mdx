## Description

This command saves UI layout preferences.
State includes panel dimensions, collapsed sections, and other layout settings.

## Validation Rules

- Preferences must exist for this user
- UI state must be valid JSON
- State size limits are enforced

## Emits

- `UserPreferencesUiStateUpdated` on success
