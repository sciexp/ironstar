## Description

This query returns all preference settings for a specific workspace.
Used for workspace settings pages and dashboard initialization.

## Read Model

Built from `WorkspacePreferencesInitialized`, `WorkspacePreferencesDefaultCatalogSet`, `WorkspacePreferencesDefaultCleared`, and `WorkspacePreferencesLayoutDefaultsSet` events.
