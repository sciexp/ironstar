## Description

This command removes a chart placement from a dashboard.
The underlying ChartDefinition in Analytics remains unaffected.

## Validation Rules

- Dashboard must exist and be owned by the user
- Chart must exist in the dashboard

## Emits

- `DashboardChartRemoved` on success
