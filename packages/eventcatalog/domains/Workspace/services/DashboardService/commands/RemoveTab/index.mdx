## Description

This command removes a tab from a dashboard.
The tab must be empty (contain no charts) before removal, or charts must be relocated to another tab first.

## Validation Rules

- Dashboard must exist and be owned by the user
- Tab must exist within the specified dashboard
- Tab must be empty (no charts assigned to it)

## Emits

- `TabRemoved` on success
