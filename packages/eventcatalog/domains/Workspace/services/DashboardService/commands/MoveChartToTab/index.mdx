## Description

This command reorganizes dashboard layout by moving a chart between tabs.
Optional grid coordinates allow repositioning during the move.

## Validation Rules

- Dashboard must exist and be owned by the user
- Chart must exist in the dashboard
- Destination tab must exist in the dashboard
- If source and destination are the same, command is rejected

## Emits

- `DashboardChartMovedToTab` on success
