## Description

This query returns the full layout state of a specific dashboard including all chart placements, tab organization, and grid positions.
Used to render the dashboard editor and viewer.

## Read Model

Built from `DashboardCreated`, `DashboardChartAdded`, `DashboardTabAdded`, `DashboardChartMovedToTab`, and `DashboardChartRemoved` events.
