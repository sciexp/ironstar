## Description

This query returns chart definitions from the Analytics bounded context that are available for placement on dashboards.
Acts as a Customer-Supplier integration with Analytics as the upstream supplier.

## Read Model

Built from Analytics `ChartDefinitionCreated` events.
Provides chart metadata without the full visualization configuration.
