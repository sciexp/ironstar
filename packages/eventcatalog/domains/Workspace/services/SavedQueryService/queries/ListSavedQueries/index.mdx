## Description

This query returns a list of saved queries for the authenticated user.
Results include name, dataset, and usage statistics.

## Read Model

Built from `SavedQueryCreated`, `SavedQueryRenamed`, and `SavedQueryDeleted` events.
Filtered by owner ID matching the authenticated user.
