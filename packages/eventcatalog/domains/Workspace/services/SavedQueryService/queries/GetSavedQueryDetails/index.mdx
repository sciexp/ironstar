## Description

This query returns complete information about a saved query including SQL, dataset, and metadata.
Used when loading a saved query into the editor.

## Read Model

Built from `SavedQueryCreated`, `SavedQueryRenamed`, `SavedQuerySqlUpdated`, and `SavedQueryDatasetRefUpdated` events.
