## Description

This command changes the dataset reference of a saved query.
Schema compatibility should be validated when changing datasets.

## Validation Rules

- Saved query must exist and be owned by the user
- New dataset reference must be a valid URI
- Schema diff should be computed if possible

## Emits

- `SavedQueryDatasetRefUpdated` on success
