## Description

This command modifies the SQL statement of a saved query.
Impact analysis identifies dependent dashboards.

## Validation Rules

- Saved query must exist and be owned by the user
- New SQL text must be non-empty
- Dependent dashboards should be warned

## Emits

- `SavedQuerySqlUpdated` on success
