## Overview

ChartOptions is a value object representing the complete set of rendering options for chart visualization.
This includes axis configuration, legend placement, grid layout, color palettes, and interactivity settings.

## Invariants

- Default values are provided via defaultChartOptions for zero-config rendering
- Axis configurations are optional to support charts without explicit axes (e.g., Pie)
- Grid configuration values are in pixels
- Color palette must contain at least one valid hex color code
- Interactive flag defaults to true

## Usage

- **ChartConfig**: Combines with ChartType for complete chart specification
- **Frontend visualization**: Consumes options to configure ECharts/Vega-Lite
- **Default configuration**: Provides sensible defaults for quick chart creation
