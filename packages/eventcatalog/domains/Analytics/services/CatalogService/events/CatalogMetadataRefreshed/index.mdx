## Description

This event records that the metadata for a catalog has been refreshed from the DuckLake source.
Metadata includes the list of available datasets, table counts, and schema version information.

## Trigger

Raised in response to the `RefreshCatalogMetadata` command after successfully fetching updated metadata from the catalog source.
